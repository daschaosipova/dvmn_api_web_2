# Обрезка ссылок с помощью clc.li

С помощью данного скрипта можно получать короткие ссылки из длинных исходных, а также просматривать статистику по количеству кликов у полученных коротких ссылок.

Для работы с данным скриптом необходимо зарегистрироваться [на сайте clc.li](https://clc.li/user/register) для получения ключа API. Он будет сгенерирован автоматически при регистрации.

Найти свой API  ключ можно будет по [ссылке в профиле](https://clc.li/user/developers/apikeys)

### Как установить
Скачайте файлы репозитория для локального использования.

В этой же папке создайте `.env` файл.
Ваш `.env` должен содержать строку:
```python
CLC_TOKEN=**ваш API ключ**
```

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```python
pip install -r requirements.txt
```
### Использование
Используем консольный ввод.

1. Создать короткую ссылку:
```python
python main.py https://7kingdoms.ru/wiki/%D0%91%D0%B0%D0%BB%D0%B5%D1%80%D0%B8%D0%BE%D0%BD

Короткая ссылка clc.li/yhOQd
```

2. Вывести количество переходов по кликам короткой ссылке:
```python
python main.py clc.li/yhOQd

Количество кликов по ссылке: 2
```
